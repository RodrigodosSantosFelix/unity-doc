!ブレンドツリー

Blend Tree （ブレンドツリー）は、float の[[AnimationParameters|イベントパラメータ]] にもとづいた似通ったアニメーション、との間の継続的なブレンディングに用いられます。典型的な例は、''speed'' パラメータにもとづいた''歩く'' と''走る'' のアニメーション間のブレンディングです。メカニムは歩くと走るの間の遷移がスムーズであることを保証します（アニメーションクリップが''揃っている'' ことが大事です：例えば、床に0.0のときに左足が着地し、右足が床に0.5のときに到着する、がどちらのアニメーションでも実現される、など）。もうひとつの典型的な例は左に走る、真っ直ぐ走る、右に走るアニメーションが全て ''direction'' パラメータの値、0.0の場合は左、1.0の場合は右としてその中間値、にもとづいているような場合です。

新規のブレンドツリーで作業するには：
# ~~Animator Controller Window~ 上の空いているどこかで右クリックします
# ^^From New Blend Tree^^ を選択します。
# Blend Tree をダブルクリックして ~~Blend Tree Inspector~~ を表示します

Attach:MecanimBlendTreeInitial.png

インスペクタで、最初にやるべきことはこのブレンドツリーを制御する [[AnimationParameters|Animation Parameter]] を選択することです。 

次に個々のアニメーションを追加するには^^+ -> Add Motion Field^^ をクリックして、ブレンドツリーにアニメーション クリップを追加します。終わったときは次のように表示されます：

Attach:MecanimBlendTree.png

赤い垂直バーがイベントパラメータの現在値を示します。アニメーションがどうなるかプレビューするには、 ~~Animation Preview Window~~ で^^Play^^ を押して、バーを左右にドラッグします。

良くある質問：$$State Machines$$ と $$Blend Trees$$ はどのように使い分ければ良いですか？
答え：ステートマシンは関連のない、不連続なしきい値やbooleanパラメータにもとづいた、アニメーション間の遷移で使用します。ブレンドツリーは、連続的（float）で、共通性のあるアニメーションで使用します。
