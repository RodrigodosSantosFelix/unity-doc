!Unity Web PlayerおよびUnityObject2とのブラウザコミュニケーション

ブラウザコミュニケーションは大部分において、 [[UnityWebPlayerandbrowsercommunication|UnityObject]] と同様に行われます。しかし、Unity Web Playerコンテンツの関数をウェブページからコールするのは次のように行われます：

[@
<script type="text/javascript" language="javascript">
<!--
//initializing the WebPlayer
var u = new UnityObject2();
u.initPlugin(jQuery("#unityPlayer")[0], "Example.unity3d");

function SaySomethingToUnity()
{
	u.getUnity().SendMessage("MyObject", "MyFunction", "ウェブページから、こんにちは!");
}
-->
</script>
@]
